# 21st Agents SDK - ProductHunt Demo

A simple Flask application showcasing the 21st Agents SDK product page.

## Routes

- `/` - Home page displaying product info
- `/api/product` - JSON API for product data
- `/api/vote` - POST endpoint to vote for the product (one vote per IP)
- `/health` - Health check endpoint

## Setup

```bash
pip install -r requirements.txt
python app.py
```

## Run

```bash
python app.py
```

The app will start on `http://localhost:5000`
